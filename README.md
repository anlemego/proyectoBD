Aplicación de escritorio para la gestión de un espacio de estacionamiento con clientes y usuarios, un CRUD

Se explica como está estructurado el proyecto:
- DataBase:
    Contiene los archivos BD para realizar las queries en cada tabla de la base de datos

- Utils:
    Archivos de utilidades:
        - Conexión_BD es para realizar la conexión a la base de datos
        - entidades es para definir las clases con sus getters y setters
        - helpers es para limpiar las vistas

- Views:
    Contiene los archivos que son las vistas de cada módulo

Instrucciones_Instalacion:
    Detalla las librerías usadas y cómo instalarlas

main:
    El archivo a ejecutar para correr la aplicación
